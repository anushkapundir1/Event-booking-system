## Overview
Event Booking System is a full-stack web application built using Django
to manage events and streamline user registrations.
The platform supports role-based access and provides an admin dashboard
for efficient event and user management.

This project demonstrates **end-to-end web development**,
from backend logic to responsive frontend design.



## Key Features
- Event creation, listing, and management
- Role-based authentication (Admin / User)
- Admin dashboard for event monitoring
- Secure and efficient database operations
- Responsive user interface



## Tech Stack
- Python
- Django
- HTML, CSS, JavaScript


## Learning Outcomes
- Designing full-stack web applications
- Implementing authentication and authorization
- Managing databases using Django ORM
- Creating responsive and user-friendly interfaces



## Future Enhancements
- Payment gateway integration
- REST API using Django REST Framework
- Email notifications and reminders
